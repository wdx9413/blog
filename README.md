# 个人博客的开发

演示地址：[39.97.166.150](http://39.97.166.150)

功能:

  1. 登录注册
  2. 前台浏览
      * 文章分类浏览
      * 评论、留言模块
  3. 后台
      * 文章管理
  4. 用户校验
  5. 搜索功能（未部署）

前端（front）：

* 构建工具：create-react-app
* 技术：react+ router + redux
* 富文本编辑器：react-draft-wysiwyg

后端（end）：

* 框架：springboot 2.1.3-RELEASE
* 数据库： mysql
* 安全框架： shiro（之后有时间改为spring security）

搜索（blog-search）：

* 技术： elasticsearch-6.2.2
